Giới thiệu

Roblox Game Uploader là ứng dụng web được phát triển bằng Flask, cho phép người dùng tải lên và xuất bản game Roblox với giao diện người dùng hiện đại và thân thiện.

Công nghệ
Flask Python JavaScript HTML5 CSS3 Bootstrap 5 Roblox API
Tính năng
  • Tải lên file game Roblox (.rbxl)
  • Xác thực API key an toàn
  • Hệ thống giới hạn tốc độ và token
  • Giao diện người dùng hiện đại
  • Chức năng kéo và thả file
  • Chế độ sáng/tối
  • Lưu thông tin đăng nhập
  • Tải xuống mã nguồn dự án
